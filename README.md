# World Happiness Analysis (2015–2019)

## Project Overview

This project analyzes the World Happiness Report (2015–2019) to identify the key factors influencing national happiness.

## Tools

- SQL
- Python
- Power BI
- Excel

## Files

- World Happiness documentation.docx
- World Happiness Data.png

## Key Findings

- GDP per Capita strongly correlates with Happiness Score.
- Social Support is one of the strongest predictors.
- Healthy Life Expectancy contributes significantly to national happiness.
